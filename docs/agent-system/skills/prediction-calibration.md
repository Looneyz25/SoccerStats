# Prediction Calibration Skill

Use to review and improve model probability quality.

## Steps

1. Split completed predictions by market.
2. Count hits, misses, and pending.
3. Compare model confidence with actual hit rate where probabilities exist.
4. Identify leagues or markets with poor performance.
5. Recommend one model change at a time.

## Avoid

- Optimizing only on the latest day.
- Treating raw hit rate as profit.
- Mixing missing odds with no-value picks.
